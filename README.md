# WordCounter
A small program for counting
暂不支持通配符（因为用C太麻烦，可以在递归查找函数中增加匹配条件，但是要对用户给出的路径进行处理得到文件后缀名）；
不支持带中文的代码文本（ASC码和UNICODE转化麻烦，如要支持必须重写countNum函数中的判断条件），但是支持文件中文路径；
高级功能基本实现，可供用户用windows自带界面选择文件，但是不用图形界面输出结果，一来是画出来太丑，二来是没用过插件难画得很；
统计功能直接用函数模拟出来，基于C语言，没用到正则表达式

update 2019/9/21 15:56 支持通配符，采用正则表达式匹配文件后缀名，不符合则过滤之
